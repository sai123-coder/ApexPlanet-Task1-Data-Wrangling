# Task 1 – Data Immersion & Wrangling

## Objective
The objective of this task is to understand, clean, and transform a customer credit dataset to make it suitable for further analysis.

## Dataset Description
The dataset contains customer credit information such as age, income, credit score, credit utilization, loan balance, employment status, and payment behavior over multiple months.

## Folder Structure
- data/raw: Original dataset (unchanged)
- data/cleaned: Cleaned and transformed dataset
- notebooks: Jupyter notebook used for data exploration and cleaning
- scripts: Python script for automated data cleaning
- docs: Data dictionary describing each column
- outputs: Additional outputs or summaries

## Steps Performed
1. Loaded and explored the dataset
2. Created a data dictionary
3. Assessed data quality (missing values, duplicates, inconsistencies)
4. Handled missing values using median and mode
5. Removed duplicate records
6. Standardized categorical data
7. Performed feature engineering (Age Group, High Credit Utilization)
8. Saved the cleaned dataset
